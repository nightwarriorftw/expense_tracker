# [Expense Tracker](https://expensetracker007.netlify.app)

A simple React App that calculates your daily expenses

## :wrench: Development

#### Clone the repo
```
git clone https://github.com/nightwarriorftw/expense_tracker.git
```

#### Install the requirements
```
npm install
```

#### Run
```
npm start
```
